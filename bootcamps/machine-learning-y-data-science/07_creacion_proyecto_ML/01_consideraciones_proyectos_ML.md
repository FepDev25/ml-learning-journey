# Consideraciones a tener en cuenta en proyectos de Machine Learning

- En proyectos reales se tienen conjuntos de datos complejos.
- Comprender como se deben pre procesar los datos.

## Importancia del conjunto de datos

- Muchas veces, no importa el algoritmo y si se tiene un conjunto de datos lo suficientemente grande, se obtienen buenos resultados.
- Si tenemos un conjunto de datos lo suficiente grande, la selección del algoritmo casi que queda en segundo plano.

## Conjunto de datos

- En un dataset se encontrarán diferentes tipos de datos: numéricos continuos, numéricos discretos, categoricos, etc.
- Las caracteristicas son un atributo o grupo de atributos que constituyen una propiedad particular o un conjunto de propieades que es único, medible y diferenciable.
- Un conjunto de datos vienen compuesto por:
  - m = número de ejemplos
  - n = número de características
  - y = etiqueta de clase
- Recomendaciones:
  - Disponer una cantidad suficiente de datos.
  - Escencial utilizar un conjunto de datos que sea representativo del problema que se desea resolver.
  - Los datos deben ser de calidad.
  - Debe seleccionarse un conjunto de características adecuado, minimizando el número de características irrelevantes.
